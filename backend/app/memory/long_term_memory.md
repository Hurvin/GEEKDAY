## Culture Rules
- 14:00-16:00 是潮汕午休高频时段，建议安排茶馆、展馆等轻活动。
- 节庆闹热时段应预留机动交通时间，避免核心拥堵圈。
- 美食打卡优先错峰，餐饮高峰尽量提前 30-45 分钟到店。
- 第一泡茶称“头冲”，通常直接倒掉以“洗茶”，让后续茶味更醇厚，请勿直接饮用。
- 用餐禁忌：不可敲碗、不可将筷子插在饭中（祭祀专用），进食时避免发出吧唧声。
- 正月初七习俗“七样羹”：将芥菜、百合、芹菜、大蒜、春菜、韭菜、芥蓝同煮，寓意新春发大财、久合各人。
